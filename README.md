# Gaia Hertzsprung–Russell Diagram
### Research question:
    How can Gaia DR3 astrometric and photometric measurements be used to identify different stellar populations among nearby stars?

### Background
A Hertzsprung–Russell diagram compares stars by luminosity or absolute magnitude and colour or temperature. It reveals groups such as:

- Main-sequence stars
- Red giants
- White dwarfs

Gaia provides measurements including stellar positions, parallax, proper motion, radial velocity and brightness information for a vast catalogue of astronomical sources.

### Goal
Create a Hertzsprung-Russell diagram using Gaia DR3.

The diagram should reveal the main sequence, red giants and white dwarfs.
